# 🍽️ Restaurant Sales Data Analysis

## Project Overview

This project analyzes restaurant sales data to discover customer spending behavior, revenue trends, and business insights.

The dataset includes billing information, customer demographics, meal timing, and tipping behavior.

---

## Features

- Data cleaning
- Exploratory data analysis
- Revenue analysis
- Customer behavior analysis
- Correlation analysis
- Visual insights

---

## Tech Stack

- Python
- Pandas
- Seaborn
- Matplotlib

## Analysis Performed

### Revenue Analysis
- Total revenue
- Average bill amount
- Revenue by day

### Customer Analysis
- Tips by gender
- Smoker vs non-smoker spending
- Group size spending

### Trend Analysis
- Lunch vs dinner comparison
- Revenue distribution

### Statistical Analysis
- Correlation heatmap

---

## Key Insights

- Dinner customers generate more revenue than lunch customers
- Larger customer groups spend more
- Higher bills generally result in higher tips
- Spending patterns vary by customer type
- Revenue trends differ across weekdays
